# Smart Campus Management System

## 🚀 Features
- RFID-based attendance system
- Canteen ordering with wallet integration
- Library book issue & return system
- MySQL database with triggers & procedures

## 🛠️ Tech Stack
- Java (Swing, JDBC)
- MySQL
- SQL (Triggers, Procedures)

## 📡 How it Works
RFID card → maps to PRN → performs operations like attendance, payment, and book issue.

## 🎯 Modules
- Attendance
- Canteen
- Library
- Wallet

## 💡 Concepts Used
- DBMS (Normalization, Constraints)
- OOP in Java
- JDBC Connectivity
- Exception Handling
